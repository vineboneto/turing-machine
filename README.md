## Turing Machine

> ### About

Interface Graphic to use Turing Machine, made with ReactJS

> ### Requirements

- `>= v14.13.0`
- Yarn or NPM

> ### Development

```bash
  # Dependencies
  $ yarn

  # Run
  $ yarn start
```

> ### Deployment

```bash
  # Dependencies
  $ yarn

  # Static Server
  $ yarn global add serve

  # Build
  $ yarn build

  # Start Server
  $ serve -s build
```
